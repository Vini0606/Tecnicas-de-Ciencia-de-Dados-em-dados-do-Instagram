\begin{resumo}
Este projeto de pesquisa propõe o desenvolvimento e a validação de uma metodologia híbrida para analisar comentários das páginas de instagram dos governadores do Brasil. A abordagem integra duas técnicas de Processamento de Linguagem Natural (PLN): análise de sentimentos, para identificar polaridades e emoções; modelagem de tópicos, para extrair os temas centrais das discussões. Além disso, será utilizado um framework de clusterização, para agrupar as publicações por características semelhantes. Os objetivos específicos incluem a estruturação de um pipeline de processamento textual, a aplicação dessas técnicas de forma integrada e validação dos resultados. Espera-se que esta pesquisa contribua com um modelo analítico inovador, capaz de fornecer uma compreensão mais profunda e multifacetada do sentimento e dos tópicos em grandes volumes de dados não estruturados de mídias sociais, demonstrando as vantagens de uma abordagem sinérgica na pesquisa de opinião e de mercado.

\end{resumo}
